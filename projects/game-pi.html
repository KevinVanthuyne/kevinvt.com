_GamePi

Aug 2018 (Ongoing)

After the success of my first RetroPie project, the Tronkade, I had the idea to make a smaller portable version. The main ingredients of this game console are a Raspberry Zero, a 5 inch HDMI screen, a 10.000 mah powerbank and a cheap USB SNES controller.

The project sadly isn't completed yet. The internals are basically finished, only the case still needs to be designed and 3D printed. This will be quite a challenge as there are a lot of parts that need to be accounted for.
